                       netwag : network toolbox
                       ------------------------

Available files are : 

./doc/presentation.txt :
Introduce netwag.

./INSTALL.TXT :
Describe installation (it's the FIRST thing to do).

./doc/probleminstall.txt :
Describe solutions to installation problems.

./doc/installspecial.txt :
Explain how to install for non root users, or for distributions.

./doc/problemusage.txt :
Describe solutions to usage problems.

./doc/problemreport.txt :
Explain how to report a problem.

./doc/changelog.txt :
List changes between versions.

./doc/credits.txt :
Thanks contributors.

./doc/todo.txt :
List evolutions or changes to do.

/doc/netwag.1 :
Man page of netwag (note : it is not complete).
If you seek documentation, read netwag-5.39.0-doc_html.tgz.
You can enter "man netwag" (or "man ./netwag.1", 
"nroff -man ./netwag.1 | more" if netwib is not yet installed) 
to obtain help for each function.
